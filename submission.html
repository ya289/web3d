What This 3D App Does

EsrCola 3D Exploration is an interactive web application that showcases 3D models of beverage containers. The application allows users to:

  • View detailed 3D models of three beverage containers (Coke Can, Dr Pepper Bottle, and Sprite Bottle)
  • Interact with the 3D models through rotation, zooming, and panning
  • Toggle between wireframe and solid rendering modes
  • View product information and descriptions that update dynamically when interacting with each model
  • Experience animations triggered by user interaction
  • Browse a gallery of related product images

Technical Issues & Limitations

Browser Compatibility

The X3DOM implementation works best in Chrome, Firefox, and Edge. Some features may have limited functionality in Safari and mobile browsers.

Model Loading Time

3D models may take longer to load on slower connections. A loading indicator has been implemented to improve user experience.

Mobile Performance

While the site is responsive, complex 3D rendering may affect performance on older mobile devices. The app automatically reduces rendering quality on mobile.

How to Run & Test

Setup & Testing Instructions
  1. Download the complete project from the GitHub repository
  2. Extract all files if downloading as a ZIP
  3. Important: This project requires a local server to run properly. The recommended method is using Node.js with the live-server package:
    • Ensure you have Node.js installed
    • Open a terminal/command prompt in the project folder
    • Run the command: npx live-server
    • The project will automatically open in your default browser
  4. Alternatively, you can use any other local development server like XAMPP, WAMP, or Python's built-in HTTP server
  5. Navigate to the Models page to test interactive 3D functionalities
  6. Try the following test cases:
    • Switch between different 3D models using the model selector
    • Toggle wireframe mode on and off
    • Rotate models using mouse drag or touch gestures
    • Zoom in/out using mouse wheel or pinch gesture
    • Click on models to trigger animations
    • View product descriptions that update with each model

Note: Opening the HTML files directly without a server may cause X3DOM models and some features to not work correctly due to CORS restrictions.
